# Musdoc

Application Android (Kotlin) de musicothérapie.

## Features
- Parcours émotionnel (stress, anxiété, dépression…)
- Lecteur audio intégré
- Favoris
- Catalogue musical local

## Stack
- Kotlin
- ViewBinding
- MVVM light
- SQLite

## Build

```bash
./gradlew assembleDebug
```

Ouvrir aussi directement avec Android Studio.
